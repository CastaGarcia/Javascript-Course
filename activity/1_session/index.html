    se estan importando funciones hacia un fichero, en el cual se ejecutara una sola ves,
    a.- SCRIPT
        1.- Se ejecutan una sola vez cuando el agente de usuario carga el documento y tambien 
    son ejecutados cada vez que ocurre un determinado evento
        2.- El elemento HTML Script, se utiliza para insertar o hacer referencia a un script ejecutable dentro de un 
        documento HTML o XHTML.
        3.- Los scripts pueden acompañar a un formulario para procesar los datos a medida que éstos se introducen
    b.- SCRIPT MODULE
        1.- los modulos importan y exportan funciones que pueden ser utilizadas en otros achivos o ficheros
        2.- Un módulo puede contener una clase o una biblioteca de funciones para un propósito específico.
        3.- los módulos esperan hasta que el documento HTML esté completamente listo (incluso si son pequeños 
        y cargan más rápido que HTML), y luego lo ejecuta.
        4.- se mantiene el orden relativo de los scripts: los scripts que van primero en el documento, se ejecutan primero.
